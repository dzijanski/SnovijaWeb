Часта задаваныя пытаньні

Што гэта за сайт?

Сновія — гэта архіў аматарскай супольнасці перакладчыкаў, якія працуюць над узбагачэннем беларусізацыі, галоўным чынам модаў Minecraft. Мы выкарыстоўваем сучасныя падыходы да перакладу, такія як адаптацыі і словаўтварэнне.

Як пачаць перакладаць?

Перад тым, як пачаць спампоўваць файлы, паспрабуйце пашукаць Crowdin праекта, ён часта публікуецца на серверах Discord праекта або непасрэдна на старонках модаў.

Што рабіць, калі ёсць Crowdin, але няма нашай мовы?

  1. Створіть тему на Crowdin де англійською мовою представтесь, напишіть що бажаєте перекласти їх мод рідною мовою і що це дуже важливо для вас. Обов'язково зазначте свій рівень англійської (Beginner, Elementary, Intermediate, Upper-Intermediate, Advanced) та української (ви скоріш за все Native Speaker).
  2. Вы таксама можаце зайсці на сервер праекта ў Discord і спытаць там, ці можна неяк перакласці іх мод.

Заўсёды будзьце гатовыя да таго, што распрацоўшчыкі могуць адмовіць альбо праігнараваць.

Crowdin не мае праекта, што далей?

  1. Спампуйце мод з Modrinth або CurseForge
  2. Адкрыйце файл мода з дапамогай архіватара
  3. Пойдзем гэтым шляхам assets → [Тэчка з назвай мода] → lang
  4. Капіюем на працоўны стол файл en_us.json (перакладаем выключна з ангельскай, альбо з мовы арыгінала, калі мы яе ведаем)
  5. Пасля перакладу захавайце файл як be_by.json

Мы можам адразу перакласці матэрыялы праз тэкставы рэдактар (VSCode, Sublime, звычайны Notepad), альбо мы можам выкарыстоўваць Crowdin для перакладу.

Што рабіць, калі распрацоўшчык не пакінуў файл перакладу?

Здараецца і так, што распрацоўшчык можа не пакінуць файл з перакладам, у гэтым нам дапаможа мод ItemLog. Ён адлюстроўвае ўсе існуючыя прадметы з гульні ў выглядзе ключоў да гульнявой кансолі, нам трэба толькі захаваць файл як en_us.json і гэта будзе наш зыходны код з ключамі.

Як карыстацца Crowdin?

Платформа мае абмежаванне ў 60 000 слоў для бясплатных акаўнтаў і 100 000 для праектаў з адкрытым зыходным кодам.

  1. Увайдзіце або стварыце ўліковы запіс (вы можаце выкарыстоўваць Google, Facebook, Github або іншыя)
  2. Стварыце новы праект на старонцы вашага профілю
  3. Праходзім простую наладу
  4. На старонцы праекта абярыце «Крыніцы»
  5. Спампаваць распакаваныя файлы
  6. Мы наладжваем файл, калі сайт запытвае яго (пакуль што такое сустракалася толькі з csv)
  7. На ўкладцы «Панэль кіравання» абярыце «Беларуская»
  8. Выберыце файл, які вы хочаце перакласці
  9. Перакладаем
  10. Пасля перакладу на ўкладцы «Пераклады» націсніце «Стварыць і загрузіць»

Як дадаць пераклад?

На гэтым этапе могуць узнікнуць праблемы, бо не ўсе распрацоўшчыкі прымаюць пераклады ад супольнасці.

Калі гэта праект з адкрытым зыходным кодам на GitHub, вы можаце адправіць свой пераклад як Pull Request.

Як стварыць запыт на злучэнне праз вэб-інтэрфейс?

  1. Перайдзіце на старонку праекта GitHub (ён часта замацаваны на CurseForge і Modrinth як Source Code або Issue).
  2. Трэба зрабіць форк праэкту націснув на кнопку Fork на старонцы праекта. Такім чынам мы ствараем копію праекта на нашым GitHub.
  3. Наступным крокам будзе загрузка файла перакладу ў форк па наступным шляху, стварыўшы адпаведныя тэчкі assets → [Тэчка з назвою мода] → lang (Вы можаце ўбачыць назву тэчкі ў jar-файле мода, адкрыўшы яго архіватарам).
  4. Пасля адпраўкі каміта вярніцеся ў зыходны рэпазітар і стварыце Pull Request у рэпазіты праекта.

Як стварыць запыт на злучэнне з дапамогай каманд

  1. Зрабіце форк
  2. Кланіруйце форк рэпазітара да сябе git clone url
  3. Дадайце файл перакладу ў адпаведную тэчку
  4. Дадаць пераклад да коміту git add path/to/translate
  5. Зрабіць commit git commit -m "Added belarusian translate"
  6. Адправіць у свой рэпазітар git pull
  7. Зрабіць запыт на зліццё (pull request) у рэпазітар праекта

Дзякуй за прадастаўленую інфармацыю:

Што рабіць, калі распрацоўшчык не прымае пераклад?

Менавіта для гэтага і існуе наш праект. З дапамогай рэсурснага пакета мы даем усім карыстальнікам магчымасць гуляць з беларускай мовай.

Калі вы ўжо пераклалі і хочаце падзяліцца перакладам, але распрацоўшчык не прымае пераклад, вы можаце адправіць Pull Request у наш праект SnovijaMods, а затым запоўніце форму на сайце ў раздзеле «Дадаць пераклад», падайце ўсю вядомую вам інфармацыю і адпраўце яе, мы разгледзім заяўку і дадамо ваш пераклад на сайт.

Вядома, вы можаце не адпраўляць запыт на злучэнне (Pull Request), а адправіць яго ў выглядзе файла праз зручны для вас сэрвіс абмену файламі, і мы дадамо яго самі.

Як гэта працуе?

Спампоўванне карыстальнікам з Modrinth, CurseForge альбо GitHub рэсурспаку, усталёўваецца ў тэчку з рэсурс-пакам у корані гульні і выбірае наш архіў, перацягваючы яго ўверх. Гэта прымяняе файлы лакалізацыі да ўсіх модаў у рэсурс-паке.
